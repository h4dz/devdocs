GET teams

List all the teams

Resource URL 

GET https://subdomain.pagerduty.com/api/v1/teams

Parameters

This action is paginated. See the pagination documentation for details.
Name Type Required Description
query String No Filters the result, showing only the teams whose names match the query.

Response Fields

Name Type Description
teams Array The collection of team objects returned by the query. Fields detailed below.

Team Response Fields

Name Type Description
id String The ID of the team.
name String The name of the team.
description String A description of the team.

Example

Get all teams 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/teams"

Note: Content-type of the request must be application/json
