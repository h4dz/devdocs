PUT teams/:id

Updates an existing team.

Resource URL 

PUT https://subdomain.pagerduty.com/api/v1/teams/:id

Parameters

Name Type Required Description
name String No The name of the team.
description String No A description of the team.

Team Response Fields

Name Type Description
id String The ID of the team.
name String The name of the team.
description String A description of the team.

Example

Update a team 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X PUT \
    -d '{ "name": "A Barrel of Monkeys" }' \
    "https://acme.pagerduty.com/api/v1/teams/PBAZLIU"

Note: Content-type of the request must be application/json

Response Status: 200 OK 

{
  "team": {
    "id": "PBAZLIU",
    "name": "A Barrel of Monkeys",
    "description": "Marauding across Australia",
  }
}
