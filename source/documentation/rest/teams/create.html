POST teams

Creates a new team.

Resource URL 

POST https://subdomain.pagerduty.com/api/v1/teams

Parameters

Name Type Required Description
name String Yes The name of the team
description String No A description of the team

Example

Create a team 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X POST \
    -d '{
      "name": "A Mob of Emus",
      "description": "Marauding across Australia"
    }' \
    "https://acme.pagerduty.com/api/v1/teams"

Note: Content-type of the request must be application/json

Response Status: 200 OK 

{
  "team": {
    "id": "PBAZLIU",
    "name": "A Mob of Emus",
    "description": "Marauding across Australia"
  }
}
