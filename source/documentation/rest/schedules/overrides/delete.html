DELETE schedules/:schedule_id/overrides/:id

Remove an override. You cannot remove a past override. If the override start time is before the current time, but the end time is after the current time, the override will be truncated to the current time. If the override is truncated, the status code will be 200 OK, as opposed to a 204 No Content for a successful delete.

Resource URL

DELETE https://<subdomain>.pagerduty.com/api/v1/schedules/:schedule_id/overrides/:id

Parameters

None

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/schedules/PIJ90N7/overrides/PP1565R"

Note: Content-type of the request must be application/json

Response Status: 204 No Content

Errors

CodeMessage
4001Missing or invalid 'time' parameter
4002Missing 'override' parameter
4003User Not Found
4004Invalid Override
4005Cannot Destroy Override
