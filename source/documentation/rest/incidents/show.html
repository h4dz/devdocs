GET incidents/:id

Show detailed information about an incident. Accepts either an incident id, or an incident number.

Resource URL 

GET https://<subdomain>.pagerduty.com/api/v1/incidents/:id

Example: using incident id 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/incidents/PIJ90N7"

Note: Content-type of the request must be application/json
