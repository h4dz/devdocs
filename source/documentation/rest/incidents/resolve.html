PUT incidents/:id/resolve

Resolve an incident.

Resource URL

PUT https://<subdomain>.pagerduty.com/api/v1/incidents/:id/resolve

The status of this request will be 200 if the update succeeded.

Parameters

NameTypeRequiredDescription
requester_idStringNoThe user id of the user making the request. This will be added to the incident log entry. This is only needed if you are using token based authentication.
