POST incidents/:incident_id/notes

Create a new note for the specified incident.

Resource URL 

POST https://<subdomain>.pagerduty.com/api/v1/incidents/:incident_id/notes

Parameters

Name Type Required Description
note Object Yes The note object that includes the new note parameters.
requester_id String No The user id of the user making the request. This is only needed if you are using token based authentication.

Note Parameters

Name Type Required Description
content String No The note content

Example

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X POST \
    -d '{"requester_id":"PPSFHH7","note":{"content":"New Note"}}' \
    "https://acme.pagerduty.com/api/v1/incidents/PQL04E1/notes"

Note: Content-type of the request must be application/json
