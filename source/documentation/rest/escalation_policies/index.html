Escalation Policies API

This API lets you access and manipulate escalation policies and rules.
ResourceDescription
GET escalation_policiesList existing escalation policies.
POST escalation_policiesCreate a new escalation policy.
GET escalation_policies/:idGet details about an escalation policy.
PUT escalation_policies/:idUpdate an escalation policy.
DELETE escalation_policies/:idDelete an escalation policy.
