DELETE escalation_policies/:escalation_policy_id/escalation_rules/:id

Deletes an existing escalation rule that belongs to an escalation policy. There must be a matching rule.

Resource URL

DELETE https://<subdomain>.pagerduty.com/api/v1/escalation_policies/:escalation_policy_id/escalation_rules/:id

Example

Delete an escalation policy

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/escalation_policies/PF106PL/escalation_rules/PN0ZD20"

Note: Content-type of the request must be application/json

Response Status: 204 No Content
