PUT maintenance_windows/:id

Update an existing maintenance window.

Resource URL 

PUT https://<subdomain>.pagerduty.com/api/v1/maintenance_windows/:id

Parameters

Name Type Required Description
start_time Date No The maintenance window's start time. Can only be updated on future maintenance windows. If the start_time is set to a date in the past, it will be updated to the current date.
end_time Date No The maintenance window's end time. Can only be updated on ongoing and future maintenance windows, and cannot be set to a value before start_time.
description String No Description for this maintenance window. Can only be updated on ongoing and future maintenance windows.
service_ids Array No Services that are affected by this maintenance window. Can only be updated on future maintenance windows.

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X PUT \
    -d '{
          "maintenance_window": {
            "start_time": "2012-06-16T13:00:00-04:00Z",
            "end_time": "2012-06-16T14:00:00-04:00Z",
            "description": "Description goes here",
            "service_ids": [
              "PF9KMXH"
            ]
          }
        }' \
    "https://acme.pagerduty.com/api/v1/maintenance_windows/P9UJCMM"

Note: Content-type of the request must be application/json
