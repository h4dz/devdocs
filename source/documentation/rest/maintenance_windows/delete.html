DELETE maintenance_windows/:id

Remove or cancel a maintenance window. When deleting an ongoing maintenance window, the end_time will be set to now. Future maintenance windows will be deleted. Past maintenance windows cannot be removed or cancelled.

Resource URL

DELETE https://<subdomain>.pagerduty.com/api/v1/maintenance_windows/:id

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/maintenance_windows/P9UJCMM"

Note: Content-type of the request must be application/json

Response Status: 204 No Content
