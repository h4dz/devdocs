DELETE services/:id

Delete an existing service. Once the service is deleted, it will not be accessible from the web UI and new incidents won't be able to be created for this service.

Resource URL

DELETE https://<subdomain>.pagerduty.com/api/v1/services/:id

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/services/PIJ90N7"

Note: Content-type of the request must be application/json

Response Status: 204 No Content
