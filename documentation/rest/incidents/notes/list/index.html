GET incidents/:incident_id/notes

List existing notes for the specified incident.

Resource URL 

GET https://<subdomain>.pagerduty.com/api/v1/incidents/:incident_id/notes

Example

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/incidents/PQL04E1/notes"

Note: Content-type of the request must be application/json

Response Status: 200 OK
