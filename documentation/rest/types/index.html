Types

Date and Time

All dates and times should be represented in the ISO 8601 date/time format. The time element is optional.

Example dates in ISO 8601 format:

Date and Time ISO 8601 Representation
May 6th, 2011 at 5pm UTC 2011-05-06T17:00Z
May 6th, 2011 at 3:30am PDT 2011-05-06T03:30-07
May 6th, 2011 at midnight (time is optional) 2011-05-06

Boolean Values

The following values can be used to represent true and false. These values are not case sensitive.

Value Acceptable String Representation
true 1, true
false 0, false

Object

Object types are a JSON object.

Array

Array types are used to represent a list of values.

Example array values for the include parameter on a GET request:

Values Query String Representation
contact_methods include[]=contact_methods
contact_methods, notification_rules include[]=contact_methods&include[]=notification_rules

String

String types are a sequence of UTF-8 characters.

Integer

Integer types are a number without a fractional or decimal component.
