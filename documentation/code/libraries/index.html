PagerDuty API Libraries

A number of our users have contributed open source libraries:

Name Language Description
PDJS JavaScript Javascript library to access PagerDuty REST and event APIs, written in CoffeScript.
Pygerduty Python Dropbox Python Library for PagerDuty's v1 REST API.
PagerDuty::Connection Ruby Ruby API wrapper for the PagerDuty REST API
PagerDuty event gem Ruby A library for triggering and resolving incidents.
PagerDuty event package PHP A library for interacting with PagerDuty REST API
PagerDuty Incidents for Java Java Utility for programmatically triggering and resolving PagerDuty incidents.
PdClojureBindings Clojure https://github.com/danielribeiro/PdClojureBindings
go-pagerduty Go PagerDuty API client in Go. Tested on all Go versions 1.0 and higher.
Haskell PagerDuty API Client Haskell A PagerDuty API client written in Haskell
PagerDuty.net C#/.NET A wrapper for the pagerduty.com api
PagerDuty API .NET Client C#/.NET A client for the PagerDuty Event Integration API for .NET
PagerDutyBundle PHP Trigger PagerDuty events easily from your Symfony2 application.

Please review the Disclaimer prior to downloading sample code.
