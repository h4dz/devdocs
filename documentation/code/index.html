Code written with the PagerDuty API

PagerDuty has a very active developer community, and we've gathered some of the best tools here to share:

Type Description
Libraries Libraries in Python, Ruby or JavaScript to get your project started
Tools A selection of finished tools for chat integration, reporting or scheduling PagerDuty. Also includes 2-way integrations.

Looking to try out our API? Here are some code samples to get started with:

Name Description
Ruby API Examples Comprehensive API examples in Ruby by the PagerDuty support team
Python API Examples Comprehensive API examples in Python by the PagerDuty support team
PHP API Examples Comprehensive API examples in PHP by the PagerDuty support team
PagerDutyWebhookToEmail.php PHP examples for parsing webhooks
Embeddable on-call widget Code to display an iframe with who's on call and their contact information. Live demo
On-call example Minimal example to display who is on call using PDJS
Ruby/HTTParty Minimal example of using our REST API with the Ruby gem HTTParty
pagerduty-angular An AngularJS script for filtering and modifying PagerDuty incidents.
Bloombuzz A winner of the Hack the North hackathon in Waterloo, alerts on Bloomberg stock movements in Python.
Twilio and PagerDuty A step by step tutorial to set up a phone number, via Twilio that when called will set off a PagerDuty event.
node-pagerduty A node.js client for PagerDuty.
PagerDuty Webscripts Like Zapier for coders. Make conference calls, set up heartbeat monitoring, ping HipChat, and more.
Off-Call - PagerDuty Utilities Generate report of PagerDuty activity for a specific time range.
pagerduty-alert-volume A quick command-line to get the incident volume assigned to an escalation policy broken down by week.
pagerduty_oncall How to determine who is on-call using Ruby and the PagerDuty API. For IRC.
resque-pagerduty A Resque failure backend for triggering incidents in Pagerduty.
pdresolver Watches PagerDuty for incidents triggered by other services, then resolves them as necessary. Works with Wormly.
Time On Call How much time am I on call?
pagerdutysays PagerDutySays catches a PagerDuty webhook (triggered incident) and sends the event description to a Text To Speech Program.
Grappling Grappling is a hacky domain specific language for working with PagerDuty's webhooks
pager-hours Export on call hours for all users on all schedules within a pagerduty account
jira-pagerduty-escalation_scripts This is a set of Bash and Ruby scripts that can be customized to automate the escalation of particular JIRA issues matching a JQL query to PagerDuty.
Get Incident Details with Body Get details about a specific incident or range of incidents including the incident body
